# Information Retrieval Lab

> Building Information Retrieval systems from first principles.

This repository documents my journey through the field of **Information Retrieval (IR)** by implementing core concepts, algorithms, and data structures from scratch. Rather than serving as a collection of notes, it is intended as a practical laboratory where theory is translated into working software and small-scale experiments.

The project will evolve incrementally alongside my studies, with each topic introducing new retrieval models, indexing techniques, ranking algorithms, or evaluation methods.

---

## Objectives

* Understand the foundations of Information Retrieval through implementation.
* Develop retrieval models from first principles instead of relying on existing search libraries.
* Explore the trade-offs between different indexing and retrieval strategies.
* Build a reusable codebase that gradually grows into a small search engine.
* Document design decisions, experiments, and observations throughout the learning process.

## Guiding Principles

* Prefer understanding over optimization.
* Implement algorithms from scratch whenever practical.
* Keep implementations modular and easy to extend.
* Validate concepts with experiments rather than assumptions.
* Document not only *what* was built, but *why*.

---

## Disclaimer

This repository is a learning project. While correctness and software quality are important goals, the primary objective is to deepen understanding of Information Retrieval through implementation and experimentation.

As the project grows, earlier implementations may be refactored or replaced to reflect improved understanding or better architectural decisions.
